## Level 1 — Reviews

### 1. Indian Government Press Releases

https://www.india.gov.in/news_lists

**Issues**

1. Body text appears small difficult to read for long lists
2. Link color contrast is low against white background
3. No clear hierarchy is seen
4. List items are tightly packed vertically
5. Links have weak hover/focus affordance

**Targeted Improvements**

1. Increase base font size to **16px**, titles to **18–20px**
2. Separate metadata visually (date smaller, title bold)
3. Add vertical spacing between list items

**Screenshot**

- `./screenshots/level-1/Govt-press.png.`

## Level 2 - Comparision

### Pair 1: Flipkart Mobiles vs Amazon Mobiles

**Comparison Focus**

- Information density
- Filters visibility
- Visual hierarchy

**Observations**

- Amazon prioritizes filters; Flipkart prioritizes promotions
- Amazon has clearer price hierarchy
- Flipkart has higher visual noise

### Pair 2: Telangana Portal vs Michigan State Portal

**Comparison Focus**

- Information architecture
- Navigation clarity
- Accessibility

**Observations**

- Michigan uses better spacing and hierarchy
